## CS6.NULL Lecture 1

`pwd`  == "print work directory"

`cd` == "change directory"

`.` == "current directory"

`..`  == "parent directory"

`cd ~` : change directory to home.

`cd -` : change directory to where you were previously in.

`man` == "manual pages"

`<` == "rewire the input for this program to be the contents of this file"

`>` == "rewire the output of the preceding program into this file"

`>>` == "Append"

`cat` == "prints the content of a file"

`|` == "take the output of the program to the left and make it the input of the  program to the right"

To cd to a directory, you must have execute permission on all parent directory of that directory and the directory itself. 

```shell
mv dotfiles.md foo.md	# rename dotfiles.md to foo.md
mv foo.md dotfiles.md 	# 
cp dotfiles.md ../food.md 	# copy dotfiles.md to parent directory and rename to food.md
rm ../test2.cpp	# delete the test2.cpp at parent directory
rmdir # remove an "empty" directory
mkdir "My Photos"	# make a directory called 'My Photos'
mkdir My\ Photo
man ls	# show manual pages of 'ls', press 'q' to quit pages
echo hello > hello.txt	# write "hello" to hello.txt 
cat < hello.txt	# prints the content of hello.txt
cat < hello.txt > hello2.txt # get content of hello.txt and write them to hello2,txt
cat < hello.txt >> hello2.txt # what's in hello2.txt will be "hellohello"
ls -l \ | tail -n1	# print the last line of "ls -l"
ls -l \ | tail -n1 > ls.txt # write the last line of "ls -l" to ls.txt

```
