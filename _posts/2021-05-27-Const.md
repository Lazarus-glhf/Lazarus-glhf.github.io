##Const限定符


const

```c++
    const int maxn = 512;

    maxn = 512;//error, not allowed
```

const必须被初始化

```c++
    const int i = get_size();//initialized while running
    const int j = 42;
    const int k;//error
```

**默认状态下，const对象仅在文件内有效**

使用`extrn`可以在一个文件中定义`const`并在多个其他文件中声明并使用它们。
```c++
    //in file_1.cc
    extern const int maxn = 512;

    //in file_1/h
    extern const int maxn;//与file_1.cc中定义的maxn是同一个
    //此处声明表示`maxn`的定义将在别处出现
```
